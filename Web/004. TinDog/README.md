# TinDog

## Description

- 강아지들을 위한 데이팅 앱을 소개하는 사이트 🐶
- **Tinder** 앱을 패러디하여 제작하였다.
- 브라우저의 창의 크기에 따라 레이아웃이 반응형으로 변화되도록 하였다.

## Development Information

- **Development Period** : 2023.11.03
- **Language** : HTML, CSS3, JavaScript
- **Frameworks** : Bootstrap
